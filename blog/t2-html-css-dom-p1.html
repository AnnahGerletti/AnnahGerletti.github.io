Give an analogy to describe HTML, CSS and the DOM.

How would you describe it to your non-technical friend?

These concepts can be described with a house analogy. HTML is the structure, walls rooms etc.CSS is how you styled your place, colors of walls type of carpet and so on. While DOM is how guests (aka browsers) engage with the space. It can also create floor plan or layout of the design of the house.

What is meant by boxifying design?

Boxifying means to break up a layout into smaller divisions or ‘boxes’ to make working with them easier.

What is the box model?

The box model is used when talking about design and layout. It describes how all elements have content, padding, boarders and margins.
